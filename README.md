## Overview
This project is a Java-based **Enterprise QA Automation Framework** designed to automate **UI, API, and Database testing** for web applications.
It follows **industry-standard automation practices** and demonstrates real-world QA automation capabilities used in enterprise environments.

The framework is structured using **Page Object Model (POM)** and supports **end-to-end validation** across UI, backend services, and databases.

---

## Key Features
- UI automation using **Selenium WebDriver**
- Test execution using **TestNG**
- Page Object Model (POM) for maintainable test design
- REST API automation using **REST Assured**
- Backend database validation using **JDBC**
- Modular and reusable framework structure
- CI/CD-ready for Jenkins integration

---

## Tech Stack
- **Language:** Java  
- **UI Automation:** Selenium WebDriver  
- **Test Framework:** TestNG  
- **API Testing:** REST Assured  
- **Database:** SQL Server (JDBC)  
- **Build Tool:** Maven  
- **CI/CD:** Jenkins  
- **Version Control:** Git, GitHub
